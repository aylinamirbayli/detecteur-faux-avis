# Détecteur de faux avis
Projet M1 (ETIC) : développement d’un outil qui estime si un avis en ligne est **authentique** ou **potentiellement faux** à partir de données Kaggle.

## Objectif
Coller un avis → obtenir un **score de suspicion** + une **explication**.

## Technologies
- R / RStudio
- Shiny
- Traitement de texte (NLP)
